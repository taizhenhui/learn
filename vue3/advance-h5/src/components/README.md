# 组件文档

## NavBar(导航组件)





#### 属性

| 属性名     | 含义                 | 类型      | 必填 | 默认值 |
| ---------- | -------------------- | --------- | ---- | ------ |
| isBackIcon | 是否展示左边返回icon | *Boolean* | 否   | true   |
| title      | 标题                 | *String*  | 否   | 空     |

#### 事件

| 事件名 | 含义       | 事件参数 | 参数类型 |
| ------ | ---------- | -------- | -------- |
| goBack | 返回上一页 | 无       | 无       |

#### 插槽

| 插槽名 | 含义           |
| ------ | -------------- |
| right  | 右边展示的信息 |
